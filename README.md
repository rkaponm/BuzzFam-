# BuzzFam-

BuzzFam Advertising — official website.

A fast, SEO-friendly marketing site built with [Astro](https://astro.build).

## Quick start

```bash
npm install
npm run dev        # http://localhost:4321
```

## Scripts

| Command           | Action                                          |
| ----------------- | ----------------------------------------------- |
| `npm run dev`     | Start the local dev server with hot reload      |
| `npm run build`   | Build the production site to `dist/`            |
| `npm run preview` | Preview the built site locally                  |
| `npm run check`   | Run `astro check` (TypeScript/template linting) |

## Project layout

```
src/
  layouts/    # page shell + global styles
  components/ # reusable UI (Header, Footer, …)
  pages/      # file-based routes (index.astro → /)
public/       # static assets served as-is
```
